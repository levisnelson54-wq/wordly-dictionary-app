# Wordly Dictionary SPA

## Overview
Wordly Dictionary SPA is a single Page Application built using HTML, CSS, and JavaScript. It allows users to search for English words, view their meanings, Pronounanction, examples, synonyms and listen to pronounciation audio without reloading the page
The application fetches live data from free Dictionary APi and provides a responsive ,user-friendly interface.

##Features
- Search for English words
- View definitions
- View pronunciation
- Listen to pronunciation audio
- View example sentences
- View synonyms
- View source link
- Save favorite words
- Remove favorite words
- Favorites saved using localStorage
- Dark/Light Theme
- Responsive design
- Error handling
- Loading spinner
